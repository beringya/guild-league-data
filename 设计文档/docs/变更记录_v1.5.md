# v1.5 最终需求确认与变更记录

日期：2026-06-24  
状态：Final

## 本轮确认

1. 职业范围首次初始化使用同一导入文件中双方帮会的同职业样本。
2. 同职业有效样本数不少于 20 时使用 P5/P95 生成建议范围。
3. 样本不足时使用最小值/最大值并加入安全余量；极小样本显示警告。
4. 建议范围必须由管理员确认发布，发布后固定，不随新比赛自动变化。
5. 后续调整范围或权重必须创建新规则版本，历史比赛只在管理员主动重新分析时变化。
6. MVP 同时提供单场榜和多场历史总榜。
7. 历史总榜默认按场均综合分排序，支持最低参赛场次筛选。
8. 累计贡献分、参赛场次、最高分和趋势作为辅助数据；累计贡献分不作为默认排序。
9. 当前所有待确认项已关闭，可进入完整开发。
