# 待确认事项快捷清单

## 当前状态

**暂无待确认项，需求已冻结，可直接进入开发。**

## 已确认的最终默认口径

- 人头比采用 KDA：`(击败 + 助攻) / max(重伤, 1)`。
- 参团率采用 `(击败 + 助攻) / 所在帮会总击败`。
- 每职业固定 6 个分析槽位，综合分只使用权重大于 0 的指标。
- 十三个职业的正式评分权重均已确认；潮光、荒羽、龙吟参照神相指标与权重，但独立维护范围。
- 首次没有已发布职业范围时，使用当前文件双方同职业数据生成建议：`n ≥ 20` 使用 P5/P95；样本不足使用最小值/最大值并增加安全余量。
- 建议范围必须由管理员确认发布；发布后固定，不因新比赛自动漂移。
- 单场榜与多场历史总榜同时纳入 MVP。历史总榜默认按场均综合分排序，支持最低参赛场次筛选；累计贡献分仅作辅助。
- 玩家默认生成稳定随机头像，设置页支持替换。
- Docker 首次启动创建 `admin`，随机密码只显示一次并强制首次修改。

## 可配置默认值

- P5/P95 最低样本数：20；
- 小样本安全余量：10%；
- 历史总榜最低参赛场次：3；
- 上述值均可在设置页调整，不属于业务阻塞项。

## 未来新增职业

新职业必须先配置六维范围与权重，导入预览不得静默套用其他职业规则。
