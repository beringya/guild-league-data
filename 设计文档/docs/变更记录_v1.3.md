# v1.3 需求确认与变更记录

日期：2026-06-24

## 本轮确认

1. “人头比”统一采用 KDA：`(击败 + 助攻) / max(重伤, 1)`。
2. K/D 不再作为主展示、默认排序或默认六维补足指标。
3. 重伤为 0 时使用分母 1 保存有限 KDA，并显示“未重伤”标签。
4. 个人详情新增完整六维数据分析：本人、本帮同职业平均、对手同职业平均、职业范围、标准化得分、权重和单项贡献。
5. 每个职业固定六个可配置分析轴；权重为 0 的轴只展示、不参与综合排名。
6. 设置页支持配置每个轴的数据来源、名称、范围、方向和排名权重。
7. Docker 首次运行自动创建 `admin` 并随机生成初始密码，只在首次日志显示一次。
